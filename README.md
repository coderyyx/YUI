#MonkeyUi

## Install

```bash
npm install monkeyui --save
```

## Usage

```jsx
import { DatePicker } from 'monkeyui';
ReactDOM.render(<DatePicker />, mountNode);
```

And import style manually:

```jsx
import 'monkeyui/dist/monkeyui.css';  
```
